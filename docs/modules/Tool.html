first(...)
Get the first non-nil value. For when you can't use return a or b because false is a valid return value.

Parameters:

  • ... AnyValue It is an error if not at least one value given is NotNil.

Returns:

    NotNil The first value that was not nil. Can return boolean false if that was the first applicable value.
