make_universal_circuit_connectors(definitions)
(function). Erlib Syntactic Sugar.
Calls circuit_connector_definitions.create using the Wube.universal_connector_template, and unpacks the results.

Variation number is based on the sequence in the template graphics file. Starting at 1 in the upper left corner and incrementing left-to-right, top-to-bottom.

Parameters:

  • definitions table

Returns:

  1. table Prototype/Entity.circuit_connector_sprites
  2. table Prototype/Entity.circuit_wire_connection_point

Usage:

    local sprites, points = Wube.make_universal_circuit_connectors{{
  variation     = 26,
  main_offset   = Wube.Util.by_pixel(7, 15.5),
  shadow_offset = Wube.Util.by_pixel(7, 15.5),
  show_shadow   = false,
  }}

local my_prototype = {}
my_prototype.circuit_connector_sprites     = sprites
my_prototype.circuit_wire_connection_point = points
