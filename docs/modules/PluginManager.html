Module PluginManager

Description

Module Status: Deprecated, use PluginManagerLite.

Usage:

    local PluginManager = require('__eradicators-library__/erlib/factorio/PluginManager')()

Creation

new_manager(config) Creates a new PluginManager object.
get_manager(manager_name) Fetches a previously created PluginManager object.

Class PluginManager

PluginManager:make_env(plugin_config) Creates a new plugin environment.
PluginManager:get_env(plugin_name) Fetches a previously created plugin environment.

Managed Environment

plugasset(file_path) Expands a relative path to a full path inside the managers asset_root.
Savedata Automatic global savedata management.

Shared Environment

SharedEnvironment Every manager also has a built-in anonymous shared environment for plugins that do not need a full managed environment.


Creation

new_manager(config)
Creates a new PluginManager object.

Parameters:

  • config
    • name string The name for the new manager object.
    • main_env table The default environment for all plugins that this manager manages.
    • extra_shared_env table Extra data for the shared environment.
    • asset_root string The default root path used for plugasset. (optional)

Returns:

    PluginManager A manager instance.
get_manager(manager_name)
Fetches a previously created PluginManager object.

Parameters:

  • manager_name string

Returns:

    PluginManager A manager instance.

Class PluginManager

Usage.
PluginManager:make_env(plugin_config)
Creates a new plugin environment.

Parameters:

  • plugin_config
    • PLUGIN_NAME string A unique name for this environment.
    • PLUGIN_LUA_ROOT nil or string
    • PLUGIN_ASSET_ROOT nil or string

Returns:

    ManagedEnvironment
PluginManager:get_env(plugin_name)
Fetches a previously created plugin environment. It is an error if no environment with that name exists.

Parameters:

  • plugin_name string The literal string 'SharedEnvironment' will return this managers shared environment.

Returns:

    ManagedEnvironment or SharedEnvironment

Managed Environment

plugasset(file_path)
Expands a relative path to a full path inside the managers asset_root.

Parameters:

  • file_path string
Savedata
Automatic global savedata management. Not implemented.

Shared Environment

SharedEnvironment
Every manager also has a built-in anonymous shared environment for plugins that do not need a full managed environment.

The shared environment only supports plugasset() and the default manager environment.
