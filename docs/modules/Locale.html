merge(...)
Merges n localised strings into one.

Circumvents factorios hard limitation of allowing only up to 20 subtables per localised string by creating a deeply nested table with < 20 elemenst per level.

Parameters:

  • ... LocalisedString

Returns:

    LocalisedString

Usage:

    local x = {}
for i=65, 65*2 do x[i-64] = string.char(i) end

game.print(x)
> Error: Too many parameters for localised string: 65 > 20 (limit).

game.print(Locale.merge(table.unpack(x)))
> ABCDEFGHIJKLMNOPQRSTUVWXYZ[\]^_`abcdefghijklmnopqrstuvwxyz{|}~
format_hotkey_tooltip(key, description)
Applies rich text tags to mimic vanilla hotkey tooltips. Intended to format tooltips for custom guis that can't use "__CONTROL-foobar__" localization because LuaGuiElement buttons are hardcoded.

Parameters:

  • key string The hotkey
  • description string The description

Returns:

    string A richt-text-tag decorated plain string.

Usage:

    game.print{'', Locale.format_hotkey_tooltip('Left mouse button', 'to do something awesome!')}
