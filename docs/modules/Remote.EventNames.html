Module Remote.EventNames

Numeric event names for Erlib custom events.

Requiring this file returns a table mapping {event_name -> event_number}, just like defines.events. The numbers are dynamically generated so they may change every time you start the game, change mods, etc..

Usage:

    defines.custom_events = require('__eradicators-library__/erlib/remote/shared_event_uids')

Events

on_player_changed_chunk Raised when the player moves or is teleported across a chunk border.
on_research_reset Raised when the researched state of a technology has changed from true to false.


Events

on_player_changed_chunk
Raised when the player moves or is teleported across a chunk border. This is a higher-resolution abstraction of on_player_changed_position.

Fields:

  • player_index uint
  • old_chunk ChunkPosition
  • new_chunk ChunkPosition
on_research_reset
Raised when the researched state of a technology has changed from true to false. For example because another mod called LuaForce.reset.

Fields:

  • research LuaTechnology
