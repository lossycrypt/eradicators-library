compress(arr[, i=1[, j=Array.size(arr)]])
In-place. Compresses a SparseArray into a DenseArray. The order of the elements is preserved. Can also compress only parts of the input, for example to split the compression into multiple steps.

Parameters:

  • arr SparseArray
  • i NaturalNumber Partial array start index. (default 1)
  • j NaturalNumber Partial array end index. (default Array.size(arr))

Returns:

    DenseArray
