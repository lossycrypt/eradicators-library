Concepts

ColorSpecification
DRAFT. A color specification.

There are several formats:

Hex-RGB(A):
Html-Color Name:

LuaTable-RGB(A): {r=1, g=1, b=1, a=1}

NormalizedColor
A lua table representing an RGBA color.
All values are UnitIntervals.
{r=1, g=1, b=1, a=1}

Alpha

Color(color_spec)
DRAFT. Creates a color table.

Parameters:

  • color_spec ColorSpecification

Returns:

    NormalizedColor
premultiply_alpha(rgba_color[, alpha])
Multiplies all channels of a color table with alpha. Factorio expects color in this format most of the time.

Parameters:

  • rgba_color NormalizedColor
  • alpha UnitInterval Will be used instead of color.a if given. (optional)

Returns:

    NormalizedColor A new color table.
