Eradicator's Library.

My library contains the reusable functions I use in many of my mods. This version of the library is built for Factorio 1.0.0.

The official repository is on GitHub. Latest builds are on the ModPortal.

The Factorio Api Documentation 1.0.0 and the Lua 5.2 Reference Manual are included for convenient offline reading.
