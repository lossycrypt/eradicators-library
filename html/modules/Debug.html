Module Debug

Does automatic mod-name and load-stage/phase detection.

Usage:

    local Debug = require('__eradicators-library_/erlib/factorio/Debug')()

unsafe_stage_and_phase ()
Retrieves the current load stage and phase from the filenames on the stack. In some situations like scenarios, cross loading, or metatable methods the stage/phase can not be correctly determined. In these cases nil,nil is returned. Be aware that the returned strings use _ underscore instead of - dash. Should only be used when the error throwing behavior of Debug.get_load_stage or Debug.get_load_phase is undesired.

Returns:

  1. LoadStageName or nil
  2. LoadPhaseName or nil

Usage:

    local stage,phase = Debug.unsafe_stage_and_phase()

get_load_stage ()
Gets the current LoadStageName. The stage is internally cached so these are quite fast.

Raises:

It is an error if the stage could not be detected. I.e. when calling from inside a scenario or a data stage metatable.
get_load_phase ()
