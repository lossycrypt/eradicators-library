Module Doc

This module documents the (custom) data type names used throughout this documentation.

Tables

Table A basic LuaTable without further classification.
Set A Set is a table in which all values are true and the keys are the actual A Set A Set PhysX values.
Array An Array - sometimes also called a DenseArray - is a table in which all keys are NaturalNumbers.
SparseArray A SparseArray is also a Table in which all keys are NaturalNumbers.

Numbers

number The basic Lua type for numbers.
float A Float is is any number that has a non-zero decimal fraction.
Integer Any number that does not have a decimal fraction.
NaturalNumber Any Integer >= 0.

Mixed

NotNil Any string, number, boolean, table, function or userdata.
AnyValue Any string, number, boolean, table, function, userdata or nil.
TruthyValue Any value that when negated twice evaluates to true.

Boolean

true A boolean that is always true, never false.

Load Stage / Phase

LoadStageName The name of a load stage.
LoadPhaseName The name of a load phase.
LoadStageTable A Table that contains three keys->value mappings.


Tables

Table
A basic LuaTable without further classification. Every specialized table can also be considered a normal Table. It maps NotNil keys to NotNil values.

Fields:

  • key NotNil

Usage:

    local Table = {key = value}
Set
A Set is a table in which all values are true and the keys are the actual A Set A Set PhysX values. This is useful to quickly test if a given key is in the Set or not.

Fields:

  • key true

Usage:

  • local Set = {}
for _,value in pairs(Table) do
  Set[value] = true
  end

  • if Set[key] then f() end
  • for value,_ in pairs(Set) do print(value) end
Array
An Array - sometimes also called a DenseArray - is a table in which all keys are NaturalNumbers. Additionally the sequence of numbers must be uninterrupted - for every key n in the array there must also be a key n-1, except for n=1.

Fields:

  • [1] NaturalNumber
  • [2] NaturalNumber
  • [3] NaturalNumber
SparseArray
A SparseArray is also a Table in which all keys are NaturalNumbers. Contrary to an Array the sequence of keys may be discontinuous. These are often used tostore entities indexed by their unit_number.

Fields:

  • [1] NaturalNumber
  • [5] NaturalNumber
  • [42] NaturalNumber

Usage:

    :
  local entities = {[entity.unit_number] = entity}

Numbers

number
The basic Lua type for numbers. It can be any Integer or float.
float
A Float is is any number that has a non-zero decimal fraction.

Usage:

    local float = 3.14159
Integer
Any number that does not have a decimal fraction.

Usage:

    local Integer = 42
NaturalNumber
Any Integer >= 0. Sometimes also called a PositiveInteger.

Mixed

NotNil
Any string, number, boolean, table, function or userdata.
AnyValue
Any string, number, boolean, table, function, userdata or nil.
TruthyValue
Any value that when negated twice evaluates to true. In Lua this is any string, number, true, table, function or userdata.

Usage:

  • local yes = ((not not TruthyValue) == true)
  • local yes = (TruthyValue and true)

Boolean

true
A boolean that is always true, never false.

Load Stage / Phase

LoadStageName
The name of a load stage. This one of three strings: "settings", "data" or "control".
LoadPhaseName
The name of a load phase. This is one of 7 strings: "settings", "settings_updates", "settings_final_fixes" "data", "data_updates", "data_final_fixes" or "control". Be aware that unlike the corresponding file names these strings use _ underscores instead of - dashes for ease of use.
LoadStageTable
A Table that contains three keys->value mappings.

Fields:

  • LoadStageName true, maps the LoadStageName to true
  • name string, the LoadStageName
  • any true, always maps the string "any" to true

Usage:

    if LoadStageTable.control then
  script.on_event(defines.events.on_tick,function()end)
  end
